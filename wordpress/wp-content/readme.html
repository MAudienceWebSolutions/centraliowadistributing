Central Iowa Distributing WordPress Theme Read Me

This document describes how to install and use your WordPress theme.

Summary

  1. Plugins

    1. Activate the "WooCommerce" plugin

      Go to WooCommerce » Settings and setup plugin options

    2. Activate the "Gravity Forms" plugin

      Go to Forms » Settings and setup plugin options

  2. Content Management

    1. Custom Post Types

      • Product

        The products of the site are handled through the Products custom post type.

    2. Page Templates

      • Default Template

        This is the template that will be used by all pages by default.

        The template displays page title and page content.

      • Home

        The home page is divided into three sections.

        The top section shows a image slider. Slides can be managed by "Homepage Slider" custom repeater field located under the WYSIWYG editor.

        The middle section the selected 3 featured products by "Select 3 products" custom field located in the "Featured Products" panel under the WYSIWYG editor.

        The bottom section shows a custom entries(image/title/text/link) managed by custom fields in the "Homepage Services Section" panel.

    3. Theme Options

      Go to Theme Options » General and setup theme options

      Allows you to change your logo, custom button and text in the header.

      Another options are to manage the location in the footer also and the links for the socials.

      Also there is option to add scripts to your pages sucha s Google Analytics

    4. Sidebar

      The "Home" page template uses the "Default Sidebar". The other page templates uses the "Subpage Sidebar".

      Go to Appearance » Widgets to set up the widgets that would appear in the each sidebar.

    5. Shortcodes

      • [year]

        Use this shortcode to display the current year.

      • [crb_button][/crb_button]

        Use this shortcode to display a button.

        The shortcode has two attributes: 'link' and 'target'.

        If you want the link of the button to be open in new tab just set up 'target' attribute to 'blank' (target="blank"). By default the link is open in the same window.

        Setup shortcode content for text of the button.

        [crb_button link="#"]Shop Now![/crb_button]
